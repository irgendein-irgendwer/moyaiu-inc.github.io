    Moderation Commands

  • /nuke - Deletes all messages from the past 2 weeks in the current channel
  • /userclear [1-100] [@someone#1234] - Deletes messages from the user
  • /slowmode [seconds] - Sets slowmode of the current channel

    Music Commands

  • /play [title/URL] - Play a song
  • /pause - Pause the current song
  • /resume - Continue playing
  • /filters - Filter list
  • /filter [filter] - Apply a filter
  • /np - Shows the current song playing
  • /search [something]- Search a song on YouTube
  • /loop [queue] - Loop a song/queue
  • /shuffle - Shuffle the queue
  • /skip - Skip a song
  • /queue - Shows the queue
  • /clearqueue - Clears the queue
  • /remove [queue number] - Removes a song from the queue
  • /back - Play the previous song

    Misc Commands

  • /serverinfo - Shows Serverinfo
  • /userinfo - Shows Userinfo
  • /server - Official server invite
  • /suggestion [something] - Suggest something
  • /embed - Create an embed
